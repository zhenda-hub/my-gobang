# 五子棋网页游戏

一个简单但功能完整的网页版五子棋游戏

## 功能特点

- 经典的五子棋规则
- 响应式棋盘界面
- 简洁直观的用户界面

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)

## 本地运行

1. 克隆仓库到本地
2. 直接用浏览器打开 `index.html` 文件即可开始游戏
